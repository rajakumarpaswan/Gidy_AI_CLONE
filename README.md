# Gidy_AI_CLONE
Profile Project – Full Stack Technical Assessment
Project Overview

This project is a full-stack application that recreates a Profile Page similar to the one on Gidy.ai.
The application allows users to view, edit, and save their profile information, with data stored in a PostgreSQL database.

The main goal of this project was to demonstrate end-to-end development skills, including frontend UI development, backend API creation, and database integration.

Tech Stack
Frontend

React.js

Mantine UI

JavaScript

CSS

Backend

Node.js

Express.js

Database

PostgreSQL

Features Implemented
1. Profile Information Display

Fetches profile data from the backend API

Displays user information such as:

Name

Bio

Profile details

2. Edit Profile

Users can switch to Edit Mode

Update profile information

Save changes directly to the database

3. Database Integration

Profile data is stored in PostgreSQL

Backend APIs handle:

Fetching profile data

Updating profile information

4. Responsive UI

Built using Mantine UI components

Clean and responsive design for better user experience
